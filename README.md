# .NET Expenses Tracker

A comprehensive expense tracking application built with .NET that serves as the backend API for managing and monitoring your personal or business expenses.

## ?? Related Repositories

- **Frontend (Angular)**: [AngularExpensesTracker](https://github.com/fatinabila/AngularExpensesTracker) - The Angular frontend application that consumes this API

## Features

- **RESTful API**: Comprehensive API endpoints for expense management
- **Expense Management**: CRUD operations for expense entries
- **Category Organization**: Manage expense categories
- **Data Validation**: Server-side validation and error handling
- **Database Integration**: Persistent data storage
- **CORS Support**: Cross-origin resource sharing for frontend integration
- **Authentication**: Secure API access (if implemented)

## Technology Stack

- **.NET**: Core framework for API development
- **C#**: Primary programming language
- **ASP.NET Core Web API**: Web API framework
- **Entity Framework Core**: Object-relational mapping
- **SQL Server/SQLite**: Database storage
- **Swagger/OpenAPI**: API documentation
